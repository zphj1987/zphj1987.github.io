补充

如果你发现你运行的脚本没有结果，这个不是脚本的问题，是因为没有生成pid文件，在配置文件/etc/ceph/ceph.conf当中增加:

pid_file = /var/run/ceph/$type.$id.pid

然后重启osd进程，检查生成了pid没有
