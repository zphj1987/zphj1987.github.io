如何通过iptables代理访问内网

场景

A机器能够联通内网机器，B机器能够联通A机器，但是访问不到内网机器，场景是希望通过A机器能够转发直接联通局域网内的其它机器

机器IP

内网为172.0.0.x/24

A机器为172.0.0.10/24

A机器为192.168.1.10/24

B机器IP为192.168.1.20/24
